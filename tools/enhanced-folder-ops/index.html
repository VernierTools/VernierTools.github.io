Blender add-on

Parent and Reveal Folder

Jump to a file's parent folder and reveal it in your OS file explorer, right from Blender.

Features
  • RevealOpen the folder containing your file in Explorer or Finder with one click.
  • ParentJump straight up to the parent folder without retyping paths.
  • Cross-platformBehaves consistently on Windows, macOS, and Linux.
Requirements & limits

Requirements
Requires Blender 4.2 or later. Works on Windows and macOS. Linux should work but is untested.

Limitations
Revealing a folder only works after the file has been saved once — an unsaved file has no folder to open yet.

Install
  1. Download the .py file on the right.
  2. In Blender, open Edit ▸ Preferences ▸ Add-ons.
  3. Open the ▾ menu at the top-right, choose "Install from Disk…", then pick the .py file.
  4. Turn the add-on on with its checkbox.

Install it, and stop digging through paths to find your project files.
