Blender add-on

Parent and Reveal Folder

Stop losing your place in Blender's File Browser. A dedicated Parent button jumps up one level while keeping the folder you came from selected and scrolled into view.

Features

In large folders, going back usually resets your scroll position, leaving you to hunt for the same folder all over again. This add-on fixes that:

  • Go to the parent folder instantly.
  • Automatically select the folder you came from.
  • Scroll it back into view.
  • Works in the File Browser and in file dialogs.

Simple, lightweight, and designed to make file navigation feel smoother.

Compatibility
  • Blender 4.2 or later.
  • Tested on Windows and macOS.
  • Linux is currently untested, but should work in theory.
Install
  1. Download the .py file on the right.
  2. In Blender, open Edit ▸ Preferences ▸ Add-ons.
  3. Open the ▾ menu at the top-right, choose "Install from Disk…", then pick the .py file.
  4. Turn the add-on on with its checkbox.
